Desktop sample app demonstrating OAuth in vb.net

AccountRight API - VB.NET desktop app

  • A simple company file browser
  • Use localhost to run locally
  • Add your API key and secret to demo the OAuth workflow
  • Return a list of accounts for a particular company file

Why OAuth?

The AccountRight Live API uses a couple of steps of authentication to ensure that people working with our customers company data files are trusted.

The steps are:

  • First: as a developer you require an API KEY and SECRET
  • Next: your app needs to allow the user to grant you permission to access their company files
  • Next: you need to get an Access Token (and refresh token) from the MYOB OAUTH server
  • Lastly: you use the token to request company files, which the user needs to log into individually

Requirements

  • Visual Studio 2010 with Visual Basic

Getting up and running

Firstly unzip to a folder and load up the solution file in the VBSamples.Account folder, Before even building this you can execute the program from the bin\debug folder
